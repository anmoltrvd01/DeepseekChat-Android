# DeepseekChat-Android



An AI-powered chatbot Android application built using Kotlin and DeepSeek API.



\## Tech Stack

\- Kotlin

\- MVVM

\- Retrofit

\- Coroutines



\## Features

\- Chat with AI

\- Real-time responses

\- Secure API handling



\## Setup

1\. Clone the repository

2\. Add your API key in `local.properties`

3\. Run the app.
